# IndieAppStack Performance Review

Use this skill for weekly and monthly performance reviews.

## Metrics

Track:

- Page views by article, category, and source.
- Search impressions and CTR when Search Console data is available.
- Outbound link clicks.
- Affiliate link clicks through `/go/[slug]`.
- Newsletter signups.
- Stack Finder starts and completions.
- Internal-link movement from guides to tools, categories, comparisons, and Stack Finder.

## Decisions

For each article, decide:

- Expand: good signal, needs more depth.
- Refresh: stale pricing, product, or source checks.
- Interlink: traffic exists but internal flow is weak.
- Monetize: high-intent traffic and approved relevant affiliate program.
- Leave alone: doing its job.
- Prune or merge: thin, duplicative, or no strategic value.

## Weekly Review Prompt

Answer:

1. Which articles produced meaningful reader actions?
2. Which high-impression pages need better titles or intros?
3. Which pages generated affiliate clicks without enough context?
4. Which tools deserve program outreach next?
5. Which next articles would strengthen internal links and conversions?

## Output

Return:

- Performance summary.
- Article action table.
- Affiliate action table.
- Tracking issues.
- Next sprint recommendations.
